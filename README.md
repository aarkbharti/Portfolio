## 🚀 Getting Started

Clone the repository

```bash
git clone https://github.com/your-username/your-repository.git
```

Navigate into the project

```bash
cd your-repository
```

Install dependencies

```bash
npm install
```

Run the development server

```bash
npm run dev
```

Build for production

```bash
npm run build
```

Start the production server

```bash
npm run start
```
